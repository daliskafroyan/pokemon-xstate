# XState Demo with PokeApi

![xstate graph image](./screenshot1.png)

This example shows how to xstate work with PokeAPI in Next.js. [Learn more about XState](https://xstate.js.org/).

## How to use

install `node_modules` by running

```bash
pnpm install
```
